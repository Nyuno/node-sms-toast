[토스트(Toast)](https://www.toast.com/)를 이용한 문자 메세지 전송 Client


## 기능

- 문자 메세지(SMS, LMS, 인증용 SMS) 전송, 취소

## 설치

npm 사용:

```bash
$ npm install sms-toast
```

yarn 사용:

```bash
$ yarn add sms-toast
```
